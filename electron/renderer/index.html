📖 使用帮助

          
        

        

          
🚀 快速开始

          
    
            
  1. 获取 Telegram Bot Token：在 Telegram 中搜索 @BotFather，发送 /newbot，按提示设置机器人名称和用户名，即可获得 Token
    2. 
            
  2. 配置 API 密钥：在「设置」区域填入至少一个 AI 提供商的 API 密钥（推荐 Groq，速度快且常有免费额度）
    3. 
            
  3. 启动机器人：点击「启动」按钮，等待状态变为「运行中」
    4. 
            
  4. 开始对话：打开 Telegram，搜索你的机器人用户名，发送消息即可与 AI 对话
    5.

🔧 支持的 AI 提供商 & 推荐模型

          

            

              
🟢 Groq（推荐）

              
特点：超低延迟推理，LPU 芯片加速

              
推荐模型：llama-3.3-70b-versatile, mixtral-8x7b-32768

特点：代码能力强，性价比极高

              
推荐模型：deepseek-chat, deepseek-coder

              
费用：价格亲民，按 token 计费

特点：中文理解优秀，阿里云生态

              
推荐模型：qwen-plus, qwen-max

🟣 OpenRouter

              
特点：统一 API 接入多模型，支持免费模型

              
推荐模型：qwen/qwen3-coder:free, meta-llama/llama-3-8b-instruct:free

              
费用：部分模型完全免费！

特点：NVIDIA 官方优化，企业级性能

              
推荐模型：meta/llama-3.1-70b-instruct, mistralai/mixtral-8x22b-instruct-v0.1

              
费用：有免费试用额度

            

          


          
❓ 常见问题 FAQ

          

            

              
💬 机器人没有响应消息？

              
    
                
  • 检查「启动」按钮是否已点击，状态是否为「运行中」
    • 
                
  • 确认 TELEGRAM_BOT_TOKEN 是否正确填写（无多余空格）
    • 
                
  • 确认至少一个 API 密钥已正确配置
    • 
                
  • 查看日志区域是否有红色错误信息
    • 
                
  • 尝试点击「重启」按钮重新启动服务
    • 
              

            

            

              
🔄 如何切换 AI 模型？

              
    
                
  • 方式一：在 Telegram 中发送 /model 命令查看可用模型
    • 
                
  • 方式二：在设置中修改 DEFAULT_PROVIDER 和 DEFAULT_MODEL，然后重启
    • 
              

            

            

              
🌐 端口被占用怎么办？

              
    
                
  • 默认端口是 3000，如果被占用可在设置中修改 PORT（如 3001、8080、9000）
    • 
                
  • 修改后点击「保存」，然后「重启」生效
    • 
              

            

            

              
⚡ 为什么回复速度慢？

              
    
                
  • 不同提供商速度差异大：Groq 最快，DeepSeek 次之
    • 
                
  • 模型大小也会影响速度：70B 模型比 8B 慢
    • 
                
  • 网络延迟：如果服务器在国外，可能需要代理
    • 
                
  • 建议优先使用 Groq + llama-3.3-70b-versatile 组合
    • 
              

            

            

              
💰 如何控制 API 费用？

              
    
                
  • 使用 OpenRouter 的免费模型（带 :free 后缀）
    • 
                
  • 利用各平台的新用户免费额度
    • 
                
  • 定期检查各平台的用量仪表盘
    • 
                
  • 设置预算告警（部分平台支持）
    • 
              

            

            

              
📝 聊天记录保存在哪里？

              
    
                
  • 聊天记录存储在本地数据库中（用户数据目录）
    • 
                
  • 使用 /history 命令可查看最近记录
    • 
                
  • 使用 /clear 可清除当前会话历史
    • 
                
  • 日志文件保存在 logs 文件夹中，可点击「打开日志文件夹」查看
    • 
              

            

          


          
⚙️ 高级功能

          
    
            
  • AI 个性设置：通过菜单 File → AI 个性设置，可自定义 AI 的语气风格和对用户的称呼
    • 
            
  • 主题切换：支持浅色、深色、跟随系统三种主题模式
    • 
            
  • 多语言界面：支持中文/英文界面切换
    • 
            
  • 实时日志：可实时查看机器人运行日志，便于调试问题
    • 
            
  • 离线分发：应用打包后可离线运行，无需联网安装依赖
    • 
          


          
⚠️ 注意事项 & 安全提示

          

            
    
              
  • 🔒 API 密钥安全：所有密钥保存在本地 .env 文件中，请勿将此文件分享给他人或上传到公开仓库
    • 
              
  • 💳 费用监控：各 API 平台按 token 计费，超出免费额度后会产生费用，请定期检查账单
    • 
              
  • 🔄 配置变更：修改 Provider/Model/PORT 后，必须点击「重启」才能生效
    • 
              
  • 📋 日志清理：长时间运行会产生大量日志，可定期点击「清空」按钮或手动删除 log 文件
    • 
              
  • 🌍 网络要求：需要稳定的互联网连接以访问各 AI 提供商的 API
    • 
              
  • 🛡️ Token 保护：Telegram Bot Token 泄露会导致他人控制你的机器人，如怀疑泄露请立即到 @BotFather  revoke 并重新生成
    • 
            

          


          
💡 使用技巧

          
    
            
  • ✨ 发送长文本时可以用多段消息，AI 会理解上下文
    • 
            
  • 🎯 可以让 AI 写代码、翻译文本、总结文章、回答问题等
    • 
            
  • 📌 使用 /new 开启新对话可以避免上下文过长导致的问题
    • 
            
  • 🔄 如果 AI 回答不理想，可以换一个模型试试（不同模型擅长的领域不同）
    • 
            
  • 📊 日志区域显示的 [ui] 开头的是界面操作日志，其他是机器人运行日志
    • 
            
  • 🎨 通过顶部主题按钮可随时切换明暗主题，保护眼睛
    •
